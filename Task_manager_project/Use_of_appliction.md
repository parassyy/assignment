 ----How to Use the Application------



1. Register New User
Go to http://localhost:3000

Click "Don't have an account? Register"

Fill in details and choose role (User/Admin)

Click Register


2. Login
Use your credentials to login

You'll be redirected to Dashboard


3. Create Tasks
Click "Create Task" button

Fill in task details

Assign to users (if admin)

Click "Create Task"


4. Manage Tasks
View all tasks in dashboard

Change task status using dropdown

Delete tasks (only admin or task creator)
 


5. Admin Features
Access to all tasks

Can assign tasks to any user

Can delete any task
